## 使用した技術
- MVVM
- Kotlin Coroutine
- Room
- Hilt
- Retrofit&OkHttp
- git-flow

### CI/CD
- GitHub Actions

## アプリの特徴
### 食べ放題のお店を探し放題
- 3000円, 5000円でフィルタリング可能
- 2回タップで探せるシンプルなUI
- 徒歩で行ける距離(1km)をの食べ放題の店を探せる

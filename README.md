- Retrofit&OkHttp

### CI/CD
- GitHub Actions

## アプリの特徴
### 食べ放題のお店を探し放題
- 数回のタップで食べ放題の店を探せるシンプルなUI
- 距離(1km, 3km, 5km)と値段(3,000円、5,000円)でリッチな検索が可能
